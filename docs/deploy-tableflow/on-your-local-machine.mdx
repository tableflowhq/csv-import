---
title: On Your Local Machine
---

This page guides you through deploying TableFlow Open Source on your local machine for testing.

### 1. Clone TableFlow
Clone the repository to retrieve the docker-compose and .env.example files
```bash
git clone https://github.com/tableflowhq/tableflow.git
cd tableflow
mv .env.example .env
docker-compose up -d
```

### 2. Embed the importer
[Follow these instructions](/embed-importer#using-tableflow-open-source) to embed the importer in your frontend with
the [React](https://tableflow.com/docs/sdk/react) or [JS](https://tableflow.com/docs/sdk/javascript) SDK, making sure
to set the `hostUrl` param to `http://localhost:3001`
